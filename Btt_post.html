How to Install This Button to your kajabi site?

We Need To Add Two Files To Install This Functionality To Our Page/Website . Watch The Video Above Or Follow The Written Guide Below !

Login To Your Website & Go to the Page Where You Want To Install This Button.

Click On Menu Icon , Click On Edit Code Option.

It Will Take You To This Page , This Is Where We Need To Install Two Simple Files

Click on Snippets Folder , Then Click On "Add a new Snippet" Option .

Name It "backtotop.liquid" & Click Create File Button.

Once the File is Created , Just Copy & Paste The Code From backtotop1 File & Save It .

Now It's Time To Add Another File . Click on Sections Folder , Then Click On "Add a new Section" Option & Name it "backtotop.liquid" , Remove Any Existing Code From It & Paste In The Code From The Other File (backtotop2) , Click Save.

It's Installed , Now You Can Customise It & You Can Start Using It .
